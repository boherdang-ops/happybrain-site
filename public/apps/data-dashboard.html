데이터를 붙여넣으면, 그래프가 됩니다

엑셀에서 복사하거나 CSV를 올리면 자동으로 분석해 막대·선·원 그래프를 그립니다. 보고서용 이미지로 바로 저장하세요.

1. 데이터 넣기

엑셀 표를 복사해서 붙여넣거나(탭 구분), CSV를 올리세요. 첫 줄은 항목 이름(머리글)로 씁니다.

모든 처리는 이 브라우저 안에서만 이뤄지고, 어디로도 전송되지 않습니다. 개인정보가 포함된 표는 올리지 마세요.

3. 그래프

이렇게 쓰세요

엑셀 표나 CSV 데이터를 넣으면 자동으로 항목·숫자 열을 인식해 그래프를 그려 줍니다. 설치·로그인 없이 브라우저에서 바로.

순서

  1. 엑셀에서 표 범위를 복사해 붙여넣거나, CSV 파일을 올립니다. (첫 줄 = 항목 이름)
  2. 📊 분석하기를 누르면 데이터 미리보기와 그래프가 나옵니다.
  3. 가로축·값·집계 방식·차트 종류를 바꿔가며 원하는 그래프를 만듭니다.
  4. 🖼 이미지(PNG) 저장으로 보고서·PPT에 바로 붙여넣으세요.

팁

  • 같은 항목이 여러 번 나오면(예: '환경'이 여러 달) 집계 방식(합계·평균)으로 묶어 보여줍니다.
  • 숫자에 콤마(1,200)나 단위(120건)가 섞여 있어도 자동으로 인식합니다.
  • 원(비중) 그래프는 항목별 비율을 볼 때 좋습니다.
⚠️ 꼭 알아두세요
  • 모든 처리는 브라우저 안에서만 이뤄집니다. 그래도 개인정보가 담긴 표는 올리지 마세요.
  • 자동 인식이 어긋나면 가로축·값 열을 직접 골라 바로잡으세요.
