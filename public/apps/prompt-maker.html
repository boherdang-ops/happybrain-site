프롬프트 도우미

하고 싶은 일을 말하면, 프롬프트가 됩니다

프롬프트 작성법을 몰라도 됩니다. 평소 말투로 "무엇을 하고 싶은지"만 적으면, AI에 그대로 넣을 지시문을 만들어 드립니다.

1. 하고 싶은 일 적기

문법·형식 신경 쓰지 마세요. 옆자리 동료에게 부탁하듯 편하게 쓰면 됩니다.

만든 프롬프트는 ChatGPT·Gemini·Claude 어디서나 그대로 쓸 수 있습니다. 개인정보는 넣지 마세요(입력 내용은 외부 AI로 전송됩니다).

2. 완성된 프롬프트

복사해서 쓰시는 AI에 붙여넣으세요. 마음에 안 들면 직접 고쳐도 됩니다.

프롬프트 작성법을 따로 배우지 않아도 됩니다. 하고 싶은 일만 평소 말로 적으면, AI가 알아듣는 지시문으로 바꿔 드립니다.

순서

  1. 왼쪽에 무엇을 하고 싶은지 편하게 적습니다. (예: "청년 유출을 막을 방법을 찾고 싶다")
  2. ✨ 프롬프트 만들기를 누릅니다.
  3. 오른쪽 프롬프트를 복사해서 ChatGPT·Gemini·Claude 등에 붙여넣습니다.
  4. 아래 이어서 이렇게 대화해 보세요를 참고해 한두 번 더 주고받으면 결과가 훨씬 좋아집니다.

잘 적는 요령 (딱 두 가지)

  • 목적을 말하세요 — "무엇을 위해" 필요한지 적으면 AI가 방향을 잡습니다.
  • 내 상황을 한 줄 — 어떤 부서, 어떤 지역, 어떤 제약(예산·기간)이 있는지 덧붙이면 훨씬 쓸모 있어집니다.
기억할 것 하나
  • 좋은 결과는 한 번에 나오지 않습니다. 첫 답을 받고 "이 부분을 더", "우리 상황엔 안 맞으니 다시"처럼 대화로 이어가는 것이 핵심입니다. 시작과 판단은 사람이, 중간 탐색은 AI가 맡습니다.
  • AI의 답은 늘 초안입니다. 사실·수치는 확인하고 쓰세요.
